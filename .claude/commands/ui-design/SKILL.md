# UI Design Guidelines Quartierfest

## Design-Prinzipien
- Modernes, cleanes Design mit viel Weissraum
- Konsistente Farbpalette (z.B. Primärfarbe Grau #555555)
- Abgerundete Ecken (border-radius: 8px)
- Subtile Schatten für Cards
- Responsive Layout

## Typografie
- Font: Inter oder System-Font
- Klare Hierarchie mit Grössen

## Komponenten-Standards
- Tabellen: gestreifte Zeilen, Hover-Effekt
- Buttons: Primär gefüllt, Danger rot, hover-Zustand
- Formulare: Floating Labels oder klare Labels mit Abstand
- Fehlermeldungen: Rot mit Icon
- Erfolgsmeldungen: Grün mit Icon
